Find out how to apply for a grant using this service. You’ll need to use the grant prospectus to help you answer the questions in the application.

Completing and submitting your application

The application form is split into sections based on how it will be assessed.

You do not have to complete it in order or all at once.

Saving progress and making changes

Your answers will not save as you type, so make sure you select ‘Save and continue’ to save any content you’ve added to a text box.

If you want to save an answer you’ve given in text box but aren’t ready to answer other questions on the page, you can simply enter some placeholder text in the other boxes and select ‘Save and continue’. You can come back and add or edit answers later.

Some people like to type their answers in Word first and then copy and paste them into the text boxes.

If you need to edit any sections, you can do this at any time even after you’ve marked a section as ‘Complete’.

Submitting your application

Once you’ve marked all sections as ‘Complete’ you’ll be given the option to submit your application.

Make sure you’re happy with all your responses before submitting as you won’t be able to edit or change anything after you’ve submitted your application.

Uploading a document

You may need to upload a document as part of your application form. Your document should be no larger than 30MB and be in an accepted file format (jpg, jpeg, png, pdf, txt, doc, docx, odt, csv, xls, xlsx, ods).

If you need to reduce the file size, you can read guidance on this depending on your file format.

The following links open in a new tab.

  • Reduce the file size of your Word documents - Microsoft Support
  • Reduce the file size of your Excel spreadsheets - Microsoft Support
  • Compress PDF Online for Adobe | Reduce PDF file size
  • Compress a picture - Microsoft Support

Getting a copy of your answers

Once you’ve submitted your application, we will email you a link to a page where you can download a copy of your answers. This will exclude any uploaded files but will still show the names of any uploaded files.

The email will come from ‘Access funding’ and will be sent as soon as you’ve submitted your application. If you cannot find the email for any reason, do not forget to check your junk or spam folder.

Contact us

If you need help using this service, contact us through our support desk.

Monday to Friday: 9am to 5pm (except public holidays).

Do not send any applications or attachments through the support desk. We'll only accept applications through this service.
