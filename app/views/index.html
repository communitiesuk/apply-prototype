This is a prototype to show the various stages of the Funding Service process.

The Funding Service is a team at the Ministry of Housing, Communities and Local Government (MHCLG) that aims to improve how funding is given to local authorities, charities, and community groups.

{% set serviceApplyHtml %}

Apply for funding

Apply for funding has been designed to allow local authorities, charities, and community groups to apply for different types of funding from Ministry of Housing, Communities and Local Government (MHCLG).

You will be able to select which funding as part of the user flow.

Assess applications for funding

The Funding Service assessment tool has been designed to help evaluate fund applications, to facilitate fund decision-making.

Here you can:

  • assess applications end to end
  • comment on specific applicant responses
  • monitor the assessment progress of a fund
{{ govukButton({ text: "Start user journey", classes: "govuk-!-margin-top-3" }) }}
{% endset %} {% set serviceDeliverFundingHtml %}

Deliver grant funding

A service for grant teams that helps them deliver and manage grant funding. This could include:

  • setting up the grant and adding information, such as grant name and type
  • designing and publishing prospectuses
  • creating forms to collect information pre-award (business case, application)
  • creating forms to collect information post-award (monitoring funding progress)
  • assessing grant applications
  • viewing monitoring data
  • payment integration with SAP
{{ govukButton({ text: "Start user journey", classes: "govuk-!-margin-top-3" }) }}
{% endset %} {% set serviceAccessFundingHtml %}

Access grant funding

A service for current and potential grant recipients to access grant funding. This could include:

  • adding information pre-award, such as an application form, benchmarking data or a business case
  • reporting on the progress of their funded programmes and projects
